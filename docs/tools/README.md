# Tools

> 최종 업데이트: 2026-06-03

이 폴더는 PolyGen 코어 생성기 주변 도구의 목적, 실행 방법, 관련 위치를 정리합니다.

## 도구 목록

| 도구 | 위치 | 문서 |
|------|------|------|
| GUI app | `gui/` | `gui.md` |
| Language Server | `polygen-lsp/` | `lsp.md` |
| VS Code extension | `polygen-vscode/` | `vscode-extension.md` |
| Schema viewer | `tools/poly-viewer/` | `poly-viewer.md` |

## 문서 원칙

- 각 도구 폴더의 `README.md`는 로컬 개발자용 명령을 짧게 유지합니다.
- `docs/tools/*.md`는 프로젝트 전체 문서 인덱스에서 참조되는 상위 설명을 담당합니다.
- 도구의 기능, 설정, 빌드 절차가 바뀌면 해당 도구 문서와 로컬 README를 함께 갱신합니다.
