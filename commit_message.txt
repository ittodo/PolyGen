feat: enum 멤버 값 할당 기능 추가 및 C# 코드 생성기 업데이트

PolyGen 스키마에서 enum 멤버에 명시적으로 정수 값을 할당할 수 있는 기능을 추가했습니다.
또한, 이 변경 사항을 반영하도록 C# 코드 생성기를 업데이트했습니다.

주요 변경 사항:
- `polygen.pest`: `enum_variant` 규칙에 선택적 정수 값 할당(`= INTEGER`)을 지원하도록 문법을 확장했습니다.
- `src/ast_model.rs`: `EnumVariant` 구조체에 `value: Option<i64>` 필드를 추가하여 파싱된 값을 저장할 수 있도록 했습니다.
- `src/ast_parser.rs`: `parse_enum` 및 `parse_inline_enum_field` 함수를 수정하여 `enum_variant`의 선택적 정수 값을 파싱하고 `EnumVariant` AST 노드에 반영하도록 했습니다.
- `src/ir_model.rs`: `EnumMember` 구조체에 `value: Option<i64>` 필드를 추가하여 IR 단계에서 enum 멤버 값을 표현할 수 있도록 했습니다.
- `src/ir_builder.rs`: `convert_enum_to_enum_def` 함수를 수정하여 AST에서 전달된 명시적 값을 `EnumMember` IR 노드에 할당하고, 값이 없는 경우 순차적으로 값을 할당하도록 로직을 구현했습니다.
- `src/csharp_generator.rs`: C# 코드 생성기를 구현하여 `enum` 정의를 생성하고, `EnumMember`의 `value` 필드를 사용하여 C# enum 멤버에 명시적 값을 반영하도록 했습니다.
